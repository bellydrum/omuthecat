# omuthecat
a fun clicking game with pictures of omu, the cat.

### dependencies

- Python3.8.9
- pip3

### build.py
Usage:
  - make sure you have Python3.8.9 installed, as well as pip3.
  - from the project root, run the following commands.
    - `virtualenv .oenv`
    - `. .oenv/bin/activate`
    - `pip install -r requirements.txt`
    - `./tools/compile`

### tools/
Contents:
  - compile.sh
  - pull.sh
